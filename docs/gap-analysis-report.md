# Fitness Log App - 페르소나별 갭 분석 보고서

## 현재 앱 기능 요약

### 핵심 기능 현황
| 기능 | 상태 | 설명 |
|------|------|------|
| 운동 기록 | 구현됨 | 세트/무게/횟수 입력, 드래그로 운동 순서 변경 |
| 운동 라이브러리 | 구현됨 | 60+ 기본 운동 (가슴/등/어깨/하체/팔/코어/유산소/HIIT/스트레칭) |
| 운동 기록 조회 | 구현됨 | 달력 뷰, 최근 운동 목록, 운동 상세 보기 |
| PR 트래킹 | 구현됨 | 운동별 개인 기록 자동 저장, 1RM 자동 계산 (Epley 공식) |
| 볼륨 통계 | 구현됨 | 주간/월간 볼륨, 부위별 세트 수, 4주 추이 차트 |
| 클라우드 동기화 | 구현됨 | Supabase 기반 백업/복원 |
| 멀티 프로필 | 구현됨 | 가족/커플 공동 사용, 프로필별 통계 분리 |
| 스마트 추천 | 구현됨 | Push/Pull/Legs 분할, 회복 상태 기반 추천 |
| 휴식 타이머 | 구현됨 | 세트 완료 후 60/90/120/180초 선택, +30초 연장 |
| Double Progression | 구현됨 | 목표 횟수 범위 설정 (근력/근비대/근지구력) |
| 웜업/드랍셋 | 부분 구현 | 데이터 구조만 있음, UI 미구현 |

## 2. 페르소나별 누락 기능 상세

### 헬창 (하드코어 트레이니)
**현재 상태**: 가장 잘 지원됨 (70%)

**누락된 핵심 기능**:
1. **주기화 프로그램**: 선형/파동/블록 주기화 프로그램 생성 및 추적
2. **고급 볼륨 분석**: 주간/부위별 볼륨 목표 설정, 과훈련 경고
3. **RPE 기록 UI**: RPE 입력 UI 및 피로도 추적
4. **세트 유형 관리**: 웜업/작업/드랍/백오프 세트 구분
5. **실패 세트 기록**: 실패 지점 기록 및 분석
6. **운동 메모/일지**: 컨디션, 수면, 영양 등 메모 기능
7. **운동 비교**: 같은 운동 다른 날짜 비교 (사이드바이사이드)
8. **내보내기**: CSV/Excel 내보내기

### 일반인 (캐주얼 운동러)
**현재 상태**: 잘 지원됨 (65%)

**누락된 핵심 기능**:
1. **동기부여 시스템**: 연속 운동 배지, 목표 달성 축하
2. **간편 기록 모드**: 원터치로 이전과 동일하게 기록
3. **운동 알림**: 운동 리마인더, 휴식일 권유
4. **주간 목표 설정**: 주 N회 운동 목표 및 진행률
5. **성취 시스템**: 마일스톤 달성 시 배지/알림
6. **소셜 공유**: 운동 완료 공유 (카카오톡 등)

### 여성 (여성 사용자)
**현재 상태**: 부분 지원 (45%)

**누락된 핵심 기능**:
1. **신체 구성 추적**: 체중/체지방률/근육량 기록 및 그래프
2. **부위별 미적 목표**: 힙업, 팔뚝살 등 목표 설정
3. **생리 주기 연동**: 생리 주기에 따른 운동 강도 조절 추천
4. **여성 친화적 운동**: 여성용 운동 프로그램/루틴
5. **칼로리 소모 추정**: 운동별 예상 칼로리 소모량
6. **사진 진행 추적**: 비포/애프터 사진 타임라인
7. **저중량 고반복 템플릿**: 여성 맞춤 기본 템플릿

### 홈트족 (홈 트레이닝)
**현재 상태**: 부분 지원 (50%)

**누락된 핵심 기능**:
1. **맨몸 운동 확장**: 변형 운동 (인클라인 푸시업, 점프 런지 등)
2. **홈 장비 필터**: 덤벨/밴드/요가매트 등 보유 장비 설정
3. **운동 설명/이미지**: 운동 자세 사진 또는 일러스트
4. **영상 가이드 링크**: YouTube 등 외부 영상 연결
5. **시간 기반 운동**: 횟수 대신 시간(초) 입력 개선
6. **서킷/HIIT 타이머**: 운동-휴식 자동 전환 타이머
7. **공간/소음 필터**: 소음 없는/좁은 공간 운동 필터
8. **일일 챌린지**: 오늘의 5분/10분 홈트 챌린지

### 초보자 (운동 입문자)
**현재 상태**: 약하게 지원 (35%)

**누락된 핵심 기능**:
1. **가이드 프로그램**: 4주/8주 초보자 프로그램
2. **운동 설명**: 각 운동의 자세한 설명 및 주의사항
3. **자세 팁**: 흔한 실수, 올바른 자세 가이드
4. **점진적 난이도**: 난이도 레벨 (초급/중급/고급) 표시
5. **격려 시스템**: 운동 완료 축하, 응원 메시지
6. **용어 해설**: 세트, 렙, 볼륨 등 용어 설명
7. **시작 무게 추천**: 체중/성별 기반 시작 무게 가이드
8. **온보딩 튜토리얼**: 앱 사용법 가이드
9. **목표 설정 마법사**: 목표 기반 프로그램 추천

### 시니어 (50+ 사용자)
**현재 상태**: 매우 약함 (20%)

**누락된 핵심 기능**:
1. **큰 폰트 모드**: 전체 UI 폰트 크기 조절
2. **고대비 테마**: 시인성 향상 색상 테마
3. **단순화 UI**: 핵심 기능만 노출되는 간소화 모드
4. **안전 경고**: 과도한 무게/횟수 경고
5. **회복 추적 강화**: 운동 후 통증/피로 기록
6. **저강도 운동 카테고리**: 시니어 맞춤 운동
7. **휴식 권장**: 충분한 휴식 시간 알림
8. **의사/트레이너 공유**: 운동 기록 PDF 내보내기
9. **음성 안내**: 음성으로 운동 안내 (접근성)
10. **간단한 네비게이션**: 뒤로가기 명확화, 실수 방지

## 4. 최종 추천 기능 목록 (Top 10)

### P0 - 즉시 구현 권장 (1-2개월 내)

#### 1. 운동 설명 및 가이드 시스템
- **영향**: 모든 페르소나 (6/6)
- **구현 난이도**: Medium
- **상세**:
  - 각 운동별 자세 설명 텍스트
  - 주요 타겟 근육 시각화
  - 흔한 실수 및 주의사항
  - 변형 운동 (쉬운/어려운 버전) 연결
- **기대 효과**: 초보자/시니어 이탈 방지, 전체 만족도 향상

#### 2. 목표 설정 및 진행률 추적
- **영향**: 모든 페르소나 (6/6)
- **구현 난이도**: Medium
- **상세**:
  - 주간 운동 횟수 목표 (예: 주 3회)
  - 월간 볼륨 목표
  - 목표 달성 시 축하 모달
  - 홈 화면에 진행률 표시
- **기대 효과**: 지속적인 운동 동기 부여

#### 3. 동기부여 및 성취 시스템
- **영향**: 5/6 (헬창 제외 전체)
- **구현 난이도**: Easy
- **상세**:
  - 연속 운동 스트릭 배지
  - 마일스톤 달성 배지 (10회/50회/100회 운동)
  - PR 달성 시 특별 애니메이션
  - "오늘도 수고했어요!" 완료 메시지
- **기대 효과**: 앱 재방문율 및 장기 사용률 증가

### P1 - 단기 구현 (2-4개월 내)

#### 4. 접근성 모드 (큰 폰트/고대비)
- **영향**: 4/6 (시니어, 초보자 집중)
- **구현 난이도**: Easy
- **상세**:
  - 설정에서 폰트 크기 조절 (작게/보통/크게/매우 크게)
  - 고대비 테마 옵션
  - 간소화 모드 (핵심 기능만 표시)
- **기대 효과**: 시니어 사용자 접근성 대폭 향상

#### 5. 신체 구성 추적
- **영향**: 4/6 (여성, 일반인, 초보자, 시니어)
- **구현 난이도**: Medium
- **상세**:
  - 체중/체지방률/근육량 입력
  - 시간별 추이 그래프
  - 목표 체중/체지방 설정
  - BMI 자동 계산
- **기대 효과**: 여성 사용자 만족도 향상, 전체 건강 관리 앱으로 포지셔닝

#### 6. 간편 기록 모드 (원터치 복사)
- **영향**: 5/6 (헬창 제외 전체)
- **구현 난이도**: Easy
- **상세**:
  - "지난번과 동일하게" 버튼
  - 템플릿에서 바로 운동 시작 (한 번 터치)
  - 자주 하는 운동 퀵 액세스
- **기대 효과**: 기록 시간 단축, 일반인/시니어 편의성 대폭 향상

#### 7. 운동 알림 및 리마인더
- **영향**: 5/6 (헬창 제외 전체)
- **구현 난이도**: Medium
- **상세**:
  - 운동 스케줄 설정 (요일/시간)
  - 푸시 알림으로 운동 리마인더
  - 3일 이상 미운동 시 격려 알림
  - 알림 on/off 설정
- **기대 효과**: 습관 형성 지원, 장기 retention 향상

#### 8. 초보자 가이드 프로그램
- **영향**: 4/6 (초보자, 홈트족, 여성, 시니어)
- **구현 난이도**: Hard
- **상세**:
  - 4주 입문 프로그램 (전신/상하체 분할)
  - 주차별 자동 스케줄
  - 운동 완료 체크 및 피드백
  - 온보딩 시 목표/경험 수준 선택
- **기대 효과**: 초보자 첫 달 이탈률 감소

### P2 - 중기 구현 (4-6개월 내)

#### 9. 홈 장비 필터 및 맞춤 운동
- **영향**: 3/6 (홈트족 집중)

- **상세**:

- **영향**: 4/6 (초보자, 홈트족, 여성, 시니어)
- **구현 난이도**: Easy
- **상세**:
  - 각 운동별 YouTube 영상 링크 (WebView 또는 외부 링크)
  - 공식 추천 영상 큐레이션
  - 사용자 영상 제보 기능
- **기대 효과**: 자세 학습 지원, 앱 내 콘텐츠 보강

## 6. 결론

현재 앱은 **헬창(고급 사용자)** 을 위한 기능은 비교적 잘 갖추고 있으나, **초보자/시니어/여성/홈트족** 을 위한 기능이 부족합니다.

### 핵심 개선 방향
1. **접근성 향상**: 큰 폰트, 간소화 UI로 시니어/초보자 진입 장벽 낮추기
2. **가이드 콘텐츠**: 운동 설명, 자세 팁으로 초보자 지원
3. **동기부여 시스템**: 배지, 스트릭, 목표로 지속적 참여 유도
4. **맞춤화**: 장비 필터, 개인 목표로 다양한 환경 지원

이 개선들을 통해 앱의 **타겟 사용자 폭을 넓히고 장기 retention을 향상**시킬 수 있습니다.
